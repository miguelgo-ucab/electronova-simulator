# ElectroNova Inc. - Simulador Empresarial

Simulador avanzado de gestión estratégica diseñado para educación ejecutiva.

## Stack Tecnológico
- **Backend:** Node.js, Express 5, MongoDB (Mongoose 9)
- **Seguridad:** JWT, BcryptJS, Helmet, Sanitización NoSQL
- **Frontend:** React 19 (Próximamente)
